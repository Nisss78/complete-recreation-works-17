
# Protoduct - プロトタイプ製品共有プラットフォーム 要件定義

## サービス概要
Protoductは、開発したプロトタイプアプリケーションをアップロードして共有するためのプラットフォームです。
製品の情報や画像を登録し、ユーザーがそれを閲覧、評価できるサービスを提供します。

## 主要機能
1. **ユーザー認証**
   - ログイン/サインアップ
   - プロフィール編集
   - 言語設定（日本語/英語）

2. **製品管理**
   - 製品登録（管理者のみ）
   - 製品詳細表示
   - 製品リスト表示（日付順/いいね数順）

3. **記事機能**
   - 記事投稿
   - 記事一覧表示
   - 記事詳細表示

4. **インタラクション**
   - 製品へのいいね
   - 製品へのコメント
   - 記事へのいいね
   - ユーザーフォロー
   - ブックマーク機能

5. **チャット機能**
   - ユーザー間メッセージ機能

## 画面構成
1. **ホーム画面**
   - 最新の製品一覧
   - 最新の記事一覧

2. **製品詳細画面**
   - 製品情報表示
   - 製品画像カルーセル
   - コメントセクション

3. **記事一覧画面**
   - 記事カード表示

4. **記事詳細画面**
   - 記事コンテンツ
   - 関連記事

5. **プロフィール画面**
   - ユーザー情報
   - 投稿した記事一覧

6. **設定画面**
   - プロフィール編集
   - 言語設定

7. **ブックマーク画面**
   - ブックマークした製品/記事一覧

## 技術スタック
- **フロントエンド**
  - React (React Router, React Query)
  - TypeScript
  - Tailwind CSS
  - shadcn/ui コンポーネントライブラリ
  - lucide-react（アイコン）
  - date-fns（日付処理）
  - recharts（チャート/グラフ）

- **バックエンド**
  - Supabase（認証、データベース、ストレージ）

- **デプロイ**
  - Vercel

## データベースモデル
1. **users**
   - Supabaseの認証ユーザー

2. **profiles**
   - ユーザーID (id)
   - ユーザー名 (username)
   - アバター画像URL (avatar_url)
   - 自己紹介 (bio)
   - SNSリンク (twitter_url, instagram_url, github_url, other_url)
   - 管理者フラグ (is_admin)
   - 言語設定 (preferred_language)

3. **products**
   - 製品ID (id)
   - 製品名 (name)
   - キャッチコピー (tagline)
   - 説明 (description)
   - アイコンURL (icon_url)
   - URL (URL)
   - 作成日時 (created_at)

4. **product_images**
   - ID (id)
   - 製品ID (product_id)
   - 画像URL (image_url)

5. **product_tags**
   - ID (id)
   - 製品ID (product_id)
   - タグ (tag)

6. **product_likes**
   - ID (id)
   - 製品ID (product_id)
   - ユーザーID (user_id)

7. **product_comments**
   - ID (id)
   - 製品ID (product_id)
   - ユーザーID (user_id)
   - コメント (content)
   - 親コメントID (parent_id)
   - 返信数 (reply_count)
   - 作成日時 (created_at)

8. **comment_likes**
   - ID (id)
   - コメントID (comment_id)
   - ユーザーID (user_id)

9. **product_bookmarks**
   - ID (id)
   - 製品ID (product_id)
   - ユーザーID (user_id)

10. **articles**
    - ID (id)
    - ユーザーID (user_id)
    - タイトル (title)
    - 内容 (content)
    - サムネイルURL (thumbnail_url)
    - いいね数 (likes_count)
    - 作成日時 (created_at)

11. **article_likes**
    - ID (id)
    - 記事ID (article_id)
    - ユーザーID (user_id)

12. **article_bookmarks**
    - ID (id)
    - 記事ID (article_id)
    - ユーザーID (user_id)

13. **follows**
    - ID (id)
    - フォローするユーザーID (follower_id)
    - フォローされるユーザーID (following_id)

## UI/UXデザイン方針
- クリーンでモダンなデザイン
- レスポンシブ対応（モバイル、タブレット、デスクトップ）
- 高級感のあるカラーパレット（深いブルー、ゴールド、ホワイトスペース）
- 洗練されたアニメーション効果
- 十分な余白を持ったレイアウト
- 読みやすいタイポグラフィ

## 高級感のある UI 要素
1. 洗練されたカラーパレット（ネイビー、ゴールド、クリーム）
2. 微妙なグラデーション効果
3. 品質の高いタイポグラフィ
4. 控えめなアニメーション
5. 余白を十分に取ったレイアウト
6. 質感のあるシャドウ効果
7. 高品質な画像表示
8. 精緻なアイコンデザイン
9. 一貫性のあるボタンスタイル
10. モノトーン+アクセントカラーの配色

## 環境変数
- SUPABASE_URL: Supabaseのプロジェクト URL
- SUPABASE_ANON_KEY: Supabaseの匿名キー
- SUPABASE_SERVICE_ROLE_KEY: Supabase管理者キー（セキュアな環境で使用）
- SUPABASE_DB_URL: Supabaseデータベース接続URL
